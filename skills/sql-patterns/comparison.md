# Comparison Pattern — Two-Month Side-by-Side

Use this pattern when the user wants to compare Month A vs Month B. Query each month separately and return both result sets so the application can compute deltas.

## Strategy

Generate TWO separate queries — one for Month A and one for Month B — using the aggregation pattern. The application will merge them and calculate changes. This is cleaner than a single pivoted query and easier for the LLM to analyze.

## Template — Month A Query

```sql
SELECT
    {dimension_columns},
    SUM(VOLUME_TONS) AS VOLUME,
    SUM(GROSS_PROFIT) AS GP,
    SUM(GROSS_PROFIT) / NULLIF(SUM(VOLUME_TONS), 0) AS MARGIN,
    SUM(WON_FLAG) AS WON,
    SUM(INQUIRY_FLAG) AS INQUIRIES,
    SUM(INQUIRY_FLAG)-SUM(WON_FLAG) AS LOST
FROM SANDBOX.ANALYTICS.SALES_ACTUALS_V
WHERE DATE_TRUNC('MONTH', DELIVERY_DATE) = '{month_a}'
GROUP BY {dimension_columns}
ORDER BY GP DESC
```

(Repeat with `'{month_b}'` for Month B.)

## Example: Customer comparison, Jan 2026 vs Feb 2026

**Month A query:**
```sql
SELECT
    COALESCE(CUSTOMER_NAME, 'Unknown') AS CUSTOMER,
    SUM(VOLUME_TONS) AS VOLUME,
    SUM(GROSS_PROFIT) AS GP,
    SUM(GROSS_PROFIT) / NULLIF(SUM(VOLUME_TONS), 0) AS MARGIN,
    SUM(WON_FLAG) AS WON,
    SUM(INQUIRY_FLAG) AS INQUIRIES,
    SUM(INQUIRY_FLAG)-SUM(WON_FLAG) AS LOST
FROM SANDBOX.ANALYTICS.SALES_ACTUALS_V
WHERE DATE_TRUNC('MONTH', DELIVERY_DATE) = '2026-01-01'
GROUP BY COALESCE(CUSTOMER_NAME, 'Unknown')
ORDER BY GP DESC
```

**Month B query:**
```sql
SELECT
    COALESCE(CUSTOMER_NAME, 'Unknown') AS CUSTOMER,
    SUM(VOLUME_TONS) AS VOLUME,

WHERE DATE_TRUNC('MONTH', DELIVERY_DATE) = '2026-02-01'

```

## Example: Multi-dimension comparison (Port + Supply Region)

**Month A query:**
```sql
SELECT
    COALESCE(PORT_NAME, 'Unknown') AS PORT,
    COALESCE(SUPPLY_REGION, 'Unknown') AS SUPPLY_REGION,
    SUM(VOLUME_TONS) AS VOLUME,
    SUM(GROSS_PROFIT) AS GP,
    SUM(GROSS_PROFIT) / NULLIF(SUM(VOLUME_TONS), 0) AS MARGIN,
    SUM(WON_FLAG) AS WON,
    SUM(INQUIRY_FLAG) AS INQUIRIES,
    SUM(INQUIRY_FLAG)-SUM(WON_FLAG) AS LOST
FROM SANDBOX.ANALYTICS.SALES_ACTUALS_V
WHERE DATE_TRUNC('MONTH', DELIVERY_DATE) = '2026-01-01'
GROUP BY COALESCE(PORT_NAME, 'Unknown'), COALESCE(SUPPLY_REGION, 'Unknown')
ORDER BY GP DESC
```
